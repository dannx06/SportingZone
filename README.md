# 🏀 SportingZone

Aplicação web desenvolvida para gerenciamento e exibição de produtos esportivos, com área administrativa para controle dos produtos cadastrados.

## 📌 Sobre o Projeto

O **SportingZone** é um sistema web criado com foco em organização, design moderno e experiência do usuário. A aplicação possui uma interface intuitiva para visualização de produtos e uma área administrativa para gerenciamento das informações.

O projeto foi desenvolvido utilizando tecnologias front-end e integração com banco de dados para armazenamento dos dados dos produtos.

# 🔐 Configuração do Supabase

O projeto utiliza o **Supabase** para armazenamento e gerenciamento dos dados.

No arquivo:

```bash
js/config.js
```

Configure suas credenciais:

```javascript
const supabaseUrl = 'SUA_URL';
const supabaseKey = 'SUA_CHAVE';
```
